## Data Flow

1. **Ingestion**: Background workers fetch jobs from external APIs (RemoteOK, Adzuna, etc.) via connectors.
2. **Normalization & Dedup**: Ingested jobs are deduplicated and normalized (company aliases, salary parsing, categories).
3. **Embeddings**: A Celery job processes new jobs and generates embeddings using GitHub Models.
4. **User interaction**: User uploads a resume; it's parsed using GPT-4o-mini, and a profile is extracted.
5. **Matching**: The backend runs a hybrid ranking (semantic search via pgvector + company diversity) to provide recommendations.
